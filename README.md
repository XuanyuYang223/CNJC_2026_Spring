# Neuron–Astrocyte Associative Memory

This repository contains materials for a journal club presentation on the paper **“Neuron–astrocyte associative memory”**.

The project focuses on understanding how neuron–synapse–astrocyte interactions can be modeled as an associative memory system, and how astrocyte processes may improve memory storage capacity.

Paper Link: https://www.pnas.org/doi/10.1073/pnas.2417788122

## Files

* `Paper Copy.pdf`
  Original paper.

* `Presentation Slide.pdf`
  Presentation slides for the journal club.

## Main Topics

* tripartite synapse model
* neuron, synapse, and astrocyte process dynamics
* Lagrangian and energy-based formulation
* connection to Dense Associative Memory
* memory storage and capacity scaling
* biological interpretation of astrocyte computation

## Reference

Kozachkov, L., Slotine, J.-J., & Krotov, D.
